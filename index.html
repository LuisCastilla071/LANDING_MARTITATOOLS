El Proyecto Académico

Misión y Planificación

Desarrollado bajo metodología Scrum, MartitaTools busca digitalizar procesos manuales para reducir pérdidas por stock estancado o mermas operativas.

  • Control centralizado de existencias.
  • Interfaz amigable para usuarios finales.
  • Base de datos en tiempo real con Supabase.
