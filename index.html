Preguntas frecuentes

¿Funciona sin internet?
Sí. Es una Progressive Web App (PWA). Funciona offline y se sincroniza cuando recuperas conexión.
¿Es gratis?
Sí, completamente gratis durante la fase beta del proyecto académico.
Acceso beta limitado

Solo estamos aceptando ferreterías para probar la versión beta. Quedan pocos cupos.
