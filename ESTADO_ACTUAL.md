# Estado del Proyecto: MartitaTools Landing Page (Edición Premium)

## 🎯 Objetivo
Landing page académica de alto nivel para MartitaApp (Control de Inventario - PWA con React + Vite + Supabase).

## 🛠️ Especificaciones de Diseño (Consensuadas)
- **Navbar:** 100% Sólida (Color `#1A365D`), con borde inferior verde `#2F855A`. Legibilidad total garantizada.
- **Hero:** Efecto **Parallax de profundidad** (fondo fijo) con **Animación Reveal Cinemática** (desplazamiento suave y desenfoque progresivo).
- **Márgenes:** Secciones con `padding: 120px 0` para una estructura despejada y profesional.
- **Contenedores:** Ancho máximo de `1200px` (`container-custom`) con márgenes de seguridad.
- **Animaciones "Ultra-Smooth":** 
    - Uso de curvas `cubic-bezier(0.16, 1, 0.3, 1)` para transiciones fluidas.
    - Efecto de **Cascada (Staggered)** en listas y beneficios mediante retardos (AOS delay).
    - **Soft Elevation:** Tarjetas que se elevan con sombras profundas y suaves al pasar el ratón.

## 🚀 Funcionalidades en la Demo (Basado en GEMINI.md)
- Sincronización en tiempo real con **Supabase**.
- Control de stock dinámico específico para ferretería.
- Gráficos interactivos de **Chart.js** (Rotación e Inventario Mensual).
